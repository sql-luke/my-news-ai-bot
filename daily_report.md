# 🤖 AI 每日新聞洞察

以下是新聞整理摘要，以及三個今日最值得關注的科技趨勢：

### 新聞摘要

近期科技新聞顯示，人工智慧（AI）已深入多個領域並帶來顯著影響。在網路安全方面，Google發布《2025廣告安全報告》，揭露其運用Gemini AI技術，已成功攔截逾99%違規廣告，全球移除超過83億則廣告，並停權逾2,490萬個帳戶；在台灣亦配合法規移除3,564個違規網址，以強化數位平台安全。

在硬體產業方面，AI應用需求激增，正加速光通訊模組從400G升級至800G，並邁向1.6T商用化，帶動矽光子及石英元件族群股價強勢上漲。同時，被動元件通路商蜜望實也受惠於AI強勁需求，3月營收創歷史新高並轉虧為盈，顯示AI對供應鏈的巨大拉抬作用。

在創意產業，知名作詞人方文山接觸AI創作，並攜手AI公司推出數位藝術展，透露曾與周杰倫討論將AI技術用於MV製作，但當時考量技術成熟度及對歌迷的責任，最終仍選擇實景拍攝。此外，軟體開發進展顯示，AI Agent已能實現自動化操控應用程式與瀏覽器，預示著更智能、自主的數位工作模式。

### 今日最值得關注的科技趨勢

1.  **AI推動基礎設施升級與硬體供應鏈成長：** AI強大的運算與資料傳輸需求，正直接驅動資料中心、光通訊模組（從400G/800G到1.6T）、矽光子及被動元件等關鍵硬體設施的升級與市場成長，相關供應鏈業者將持續受惠。
2.  **AI自動化與智能代理應用擴展：** AI不僅止於數據分析，更朝向實際操作與應用程式控制發展。從Google利用Gemini AI自動偵測並移除有害廣告，到AI Agent能操控瀏覽器與應用程式，顯示AI正逐步接管重複性任務，實現更高效的自動化管理與操作。
3.  **人工智慧與人類創作及倫理責任的融合與考量：** AI正深入藝術與創意領域，如方文山接觸AI創作並舉辦數位藝術展，預示著人機協作將成為新的創作模式。然而，同時也引發創作者對技術成熟度、作品原創性以及對受眾（歌迷）責任的思考，凸顯了在擁抱AI工具的同時，人類倫理與價值的考量仍不可或缺。
